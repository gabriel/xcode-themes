# Install (XCode 4)

 * Place files in: `~/Library/Developer/Xcode/UserData/FontAndColorThemes/`
 * Restart XCode
 * Open `XCode | Preferences | Fonts & Colors`, select theme name

# Install (XCode 3)

 * Place files in: `~/Library/Application\ Support/Xcode/Color\ Themes/`
 * Restart XCode
 * Open `XCode | Preferences | Fonts & Colors`, select theme name
